## Installation

To install KuroTranslate-Tools, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/YourUsername/KuroTranslate-Tools.git
   cd KuroTranslate-Tools
   ```

2. **Install Dependencies**:
   Ensure you have Python 3.x installed, then run:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Tool**:
   Start the application with:
   ```bash
   python main.py
   ```

## Usage

KuroTranslate-Tools provides a simple command-line interface. Here are some basic commands:

- **Disassemble a `.dat` file**:
   ```bash
   python main.py disassemble yourfile.dat
   ```

- **Parse a `.tbl` file**:
   ```bash
   python main.py parse yourfile.tbl
   ```

- **Create a Translation**:
   ```bash
   python main.py translate yourfile.tbl your_translation_file.xliff
   ```

Refer to the [documentation](#) for more advanced usage and options.

## Supported Formats

KuroTranslate-Tools supports the following file formats:

- `.tbl`: Table files containing game data.
- `.dat`: Data files used in Falcom games.
- `.xliff`: Translation interchange file format.
